Setting a Timer with Siri

Talking to a virtual assistant starts a timer

Environment

What can I do?

User Action

Saying "Hey Siri, set a timer for 3 minutes" (or a similar phrase)

System Action

A 3 minute timer starts

How do I do it?

No Action Presentation

No way to discover the action without knowing about it; impossible to know whether the exact phrase used is going to be recongnized

Before

What is going to happen?

No Preview

No way to know what will happen without triggering the action

During

Is my input working?

Input Feedback

Confirms that the user's voice is understood by transcribing the input on the screen

What is happening?

Action Feedback

Autonomous

The user has no idea whether the action will be recognized before ending the sentence

After

What happened?

Feedback

Siri says "Ok, 3 minutes and counting..." and the timer is shown on the screen

How do I go back?

Reversibility

Unrelated Action

Saying "Hey Siri, cancel the timer" will cancel it

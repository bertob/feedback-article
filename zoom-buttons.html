Zoom Buttons

Clicking the + and - buttons zooms in and out

Environment

What can I do?

User Action

Moving mouse cursor over the buttons, pressing left mouse button

System Action

Zoom level increases and decreases in fixed increments, depending on which button is clicked

How do I do it?

Action Presentation

The buttons look like all other system buttons, indicating their clickability

Before

What is going to happen?

Preview

Passive

Icons on the buttons convey whether they zoom in or out, but not how much.

During

Is my input working?

Input Feedback

Button color changes while it's being clicked

What is happening?

Action Feedback

Autonomous

Zoom animation

After

What happened?

Response

Zoom level has changed

How do I go back?

Reversibility

Unrelated Action

Pressing the other button will return the zoom level to its previous value

Setting a Timer with Siri

Starting a timer by telling Siri to do it

Environment

What can i do?

User Action

Saying "Hey Siri, set a timer for 3 minutes" (or a number of similar phrases)

System Action

A 3 minute timer starts

How do I do it?

No Action Presentation

Not advertised anywhere in the interface; it's impossible to know whether the exact phrase used is going to be recongnized

Before

What is going to happen?

No Preview

No way to know what will happen without triggering the action

During

What is happening now?

Feedback

Confirmatory

Confirms that the user's voice is understood by transcribing their command on the screen

After

What happened?

Feedback

Siri says "Ok, 3 minutes and counting..." and the timer is shown on the screen

How do I go back?

Reversibility

Effects of the action can be reversed with little effort

Unrelated Action

Saying "Hey Siri, cancel the timer" will cancel it

Ctrl+C Copying

When text is selected, pressing Ctrl+C replaces the content of the clipboard with the selected text

Environment

What can I do?

User Action

Keeping the "ctrl" key pressed while pressing the "c" key

System Action

Selected text is copied to the clipboard

How do I do it?

No Action Presentation

Not advertised anywhere in the interface

Before

What is going to happen?

No Preview

No way to know what will happen without triggering the action

During

Is my input working?

No Input Feedback

No feedback on whether the input was received

What is happening?

No Action Feedback

No information on whether the action is happening at all

After

What happened?

No Feedback

No visible effect

How do I go back?

No Reversibility

Cannot be undone, not even with a lot of effort
